# 中国历史长河图

以「长河」自上而下比喻时间流逝的**中国历史纵向时间轴**可视化页面：从传说与考古语境中的早期王朝到当代，各历史阶段在河道中以色带呈现，波纹动画象征岁月推移。

![截图](screenshot.png?raw=true)

## 功能概览

- **竖向时间河**：上古在川首、当代在川尾；向下滚动页面即「顺流而下」浏览更近的时代。
- **时间轴缩放**：左侧滑块可拉长或压缩整条时间轴（窄屏下为横向滑块），便于总览或细看某段。
- **朝代简介**：点击河中任一朝代色带，通过弹窗查看中英文简介与大致起止年代。
- **中英界面**：顶部可切换中文 / English，语言偏好会保存在浏览器本地存储中。

## 技术说明

- 单文件静态页面：`chinese-history-river.html`，内联样式与脚本，**无需构建或安装依赖**。
- 使用 SVG 绘制河道、两岸与朝代分区，辅以无障碍相关属性（如 `aria-label`）。

## 本地使用

用浏览器直接打开项目根目录下的 `chinese-history-river.html` 即可；也可由任意静态文件服务器托管该文件。

## 数据说明

页内年代为学界常用的**概略区间**，仅供通览与建立时间顺序感，不宜作为精确纪年或学术引用的唯一依据。
